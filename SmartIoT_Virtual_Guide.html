secrets.h পূরণ করুন

⏱ ৫ মিনিট
🔑
esp32/SmartIoT_v14/secrets.h
🚨
সতর্কতা! এই ফাইলটি কখনো GitHub বা কোথাও upload করবেন না। .gitignore-এ রাখা আছে।
1
File Explorer খুলুন → প্রজেক্ট ফোল্ডার → esp32 → SmartIoT_v14 → secrets.h

Notepad বা VS Code দিয়ে secrets.h ফাইলটি open করুন।

2
FIREBASE_HOST লাইনটি পরিবর্তন করুন

YOUR_PROJECT_ID এর জায়গায় আপনার আসল project URL দিন (https:// বাদ দিয়ে)।

3
FIREBASE_DB_SECRET লাইনটি পরিবর্তন করুন

Firebase Console → Project Settings → Service Accounts → Database Secrets থেকে copy করুন।

4
AES_BACKUP_KEY_HEX — নতুন key তৈরি করুন

Terminal বা Command Prompt-এ নিচের command run করুন। প্রতিটি device-এর জন্য আলাদা key।

esp32/SmartIoT_v14/secrets.h C/C++
// ⚠️ NEVER commit this file to git! #ifndef SECRETS_H #define SECRETS_H // ─── Firebase ──────────────────────────────── // Firebase Console → RTDB → copy URL (https:// বাদ দিন) #define FIREBASE_HOST "smartiot-8190a-default-rtdb.firebaseio.com" // Project Settings → Service Accounts → Database Secrets #define FIREBASE_DB_SECRET "এখানে-আপনার-real-secret-দিন" // ─── BLE Provisioning ──────────────────────── // Flutter lib/services/ble_provisioning_service.dart এর kPoP এর সাথে মিলতে হবে #define PROV_POP "Sm@rtW@t3r!BD24" // ─── AES Key — প্রতিটি device-এ unique key দিন ── // Generate: python3 -c "import secrets; print(secrets.token_hex(32))" #define AES_BACKUP_KEY_HEX "এখানে-64-char-hex-key-দিন" #endif
⚡
AES Key তৈরির command:
python3 -c "import secrets; print(secrets.token_hex(32))"
এটি run করলে একটি random 64-character key output হবে। সেটা AES_BACKUP_KEY_HEX-এ দিন।

Release Keystore তৈরি করুন

Android APK sign করার জন্য — একবারই, ভুলবেন না

⏱ ৫ মিনিট
🚨
সবচেয়ে গুরুত্বপূর্ণ সতর্কতা!
Keystore ফাইল (.jks) হারালে বা password ভুলে গেলে আর কখনো Play Store-এ ওই app-এর update দিতে পারবেন না। অবশ্যই cloud (Google Drive বা USB backup) এ রাখুন।
🗝️
৪.১ — Keystore তৈরি করুন (PowerShell)
1
PowerShell বা Command Prompt খুলুন (Admin নয়, সাধারণ)

Windows key → "PowerShell" লিখুন → Enter

2
নিচের command run করুন:
PowerShell cmd
keytool -genkey -v \ -keystore smartiot-release.jks \ -alias smartiot \ -keyalg RSA \ -keysize 2048 \ -validity 10000
3
যা জিজ্ঞেস করবে তা পূরণ করুন:

Password (মনে রাখুন!) → আপনার নাম → Bangladesh → সব তথ্য দিন

4
android/key.properties ফাইল তৈরি করুন:
android/key.properties properties

Troubleshooting

সাধারণ সমস্যা ও সমাধান

📡 ESP32 BLE scan-এ দেখাচ্ছে না ▼ বিস্তারিত দেখুন
🔴
প্রধান কারণ: Partition Scheme ভুল → BLE compile হয়নি

সমাধান:

  1. Arduino IDE → Tools → Partition Scheme → No OTA (2MB APP/2MB SPIFFS)
  2. Sketch → Verify/Compile → Upload
  3. Serial Monitor খুলুন → "[PROV] BLE provisioning started" দেখাচ্ছে কিনা
  4. Phone-এ Bluetooth ON আছে কিনা নিশ্চিত করুন
  5. Location permission দেওয়া আছে কিনা (Android BLE scan-এর জন্য দরকার)
🔥 Firebase-এ data আসছে না ▼ বিস্তারিত দেখুন
🔴
প্রধান কারণ: secrets.h-এ ভুল credentials
  1. Serial Monitor-এ "[FB] Push failed" বা HTTP error দেখুন
  2. FIREBASE_HOST: https:// নেই তো? শুধু domain থাকবে
  3. FIREBASE_DB_SECRET: Firebase Console থেকে fresh copy করুন
  4. Firebase Rules deploy করা হয়েছে কিনা
  5. ESP32 WiFi connected কিনা: OLED-এ WiFi icon দেখুন
🔨 Flutter build fail ▼ বিস্তারিত দেখুন
  1. flutter clean চালান তারপর আবার try করুন
  2. dart run build_runner build --delete-conflicting-outputs
  3. key.properties এর storeFile path সঠিক কিনা (double backslash: C:\\Users\\...)
  4. JDK version: java -version → 17+ হতে হবে
  5. flutter doctor চালান: flutter doctor -v
🚿 Pump command কাজ করছে না ▼ বিস্তারিত দেখুন
